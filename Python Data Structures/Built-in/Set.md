## Why Sets Are Important in DSA & LeetCode

Sets are extremely useful for:

- Fast membership testing â†’ O(1) average

- Removing duplicates

- Mathematical set operations (union, intersection, difference)

- Tracking visited nodes/states in BFS/DFS

- Optimizing brute force solutions (like 2Sum, 3Sum variants)
